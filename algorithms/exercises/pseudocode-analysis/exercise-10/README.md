# Ćwiczenie 6

## Opis

Dany jest następujący pseudokod, zgodny z poniższą częściową specyfikacją:

### Specyfikacja

#### Dane

* $$n$$ - liczba naturalna, $$n>0$$ 

### Pseudokod

```
1. suma := 1
2. i := 2
3. Dopóki i <= n, wykonuj:
    4. j := i
    5. p := 1
    6. Dopóki j >= 1, wykonuj:
        7. p := p * i
        8. j := j - 1
    9. suma := suma + p
    10. i := i + 1
    
11. Wypisz suma
```
