# Ćwiczenie 2

## Opis

Dany jest następujący pseudokod, zgodny z poniższą częściową specyfikacją:

### Specyfikacja

#### Dane

* $$x$$ - liczba całkowita

### Pseudokod

```
1. k := 1
2. n := 0
3. w := 0
4. Dopóki n < x, wykonuj:
    5. w := w + k
    6. k := k + 2
    7. n := n + 1
8. Wypisz w
```
