# Ćwiczenie 1

## Opis

Dany jest następujący pseudokod, zgodny z poniższą częściową specyfikacją:

### Specyfikacja

#### Dane

* $$x$$, $$y$$ - liczby naturalne, większe od $$0$$ 

### Pseudokod

```
1. q := 0
2. r := x
3. Dopóki y <= r, wykonuj:
    4. q := q + 1
    5. r := r - y
    
6. Wypisz q
7. Wypisz r 
```
