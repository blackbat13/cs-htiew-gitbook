# System binarny

## Konwersja z dziesiętnego na binarny - przykład

| **div** | **mod** |
| :-----: | :-----: |
| $$25$$  |  $$1$$  |
| $$12$$  |  $$0$$  |
|  $$6$$  |  $$0$$  |
|  $$3$$  |  $$1$$  |
|  $$1$$  |  $$1$$  |
|  $$0$$  |         |

$$
25_{10}=11001_2
$$

## Konwersja z binarnego na dziesiętny - przykład

$$
11001_2=1*2^4+1*2^3+0*2^2+0*2^1+1*2^0=2^4+2^3+2^0=16+8+1=25_{10}
$$
