# Rozwiązanie 3

## Treść zadania

Napisz program zgodny z poniższą specyfikacją.

### Specyfikacja

#### Dane

* $$a, b$$ - dwie liczby całkowite, różne od zera

#### Wynik

* Wynik dzielenia liczb $$a$$ i $$b$$ 

## Rozwiązanie

```python
a = int(input("Podaj pierwszą liczbę: "))
b = int(input("Podaj drugą liczbę: "))

wynik = a / b

print(f"{a} / {b} = {wynik}")
```
